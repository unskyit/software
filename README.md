An ultra-fast, premium desktop application for compressing and optimizing images without sacrificing quality. Built with a deeply robust UI/UX and a lightweight architecture to ensure a seamless experience.

### ✨ Key Features
* **100% Offline & Private:** Processes images entirely locally on your machine. Zero cloud uploads, ensuring total user privacy.
* **Lightweight Architecture:** Built purely with Vanilla HTML/CSS/JS on the frontend for maximum performance—no heavy frameworks attached.
* **Native Desktop Power:** Powered by a deeply optimized Rust backend for blazing-fast file handling and native OS integration.
* **Premium Glassmorphic UI:** A clean, crisp, and modern interface designed for effortless usability.

### 🚀 Getting Started

**Prerequisites:**
Ensure you have [Node.js](https://nodejs.org/) and [Rust](https://www.rust-lang.org/) installed on your machine.

**Installation:**
1. Clone the repository:
   `git clone https://github.com/yourusername/image-compressor-pro.git`
2. Install frontend dependencies:
   `npm install`
3. Run the application in development mode:
   `npm run dev`
